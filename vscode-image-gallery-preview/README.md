# Image Gallery Preview

在 VSCode 侧边栏中预览 Android、Flutter、iOS 资源文件。

## 功能说明

- 按平台、项目、模块、目录分组浏览资源
- 支持图片、音频、视频、Lottie 等资源类型
- 支持搜索、筛选、复制资源引用与 MD5
- 支持查看 MediaInfo / 图片详细信息
- 支持检测后续新添加的重复资源

## 使用方式

1. 打开侧边栏中的 `Image Gallery`
2. 等待索引完成后浏览资源
3. 点击资源项右下角按钮查看信息或复制内容

## 说明

- 音频和视频会调用系统默认应用打开
- `Sync` 用于同步新增、删除、修改
- `Refresh` 用于强制重新索引和重建元数据
